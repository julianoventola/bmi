# Features

- Main page with selectables:
  - Gender
  - Height (in a slides)
  - Weight
  - Age
- Result page calculating the BMI using just weight and height

## Getting Started

- Clone the repository
- run in cmd:
  ```cmd
  flutter pub get
  flutter run
  ```
